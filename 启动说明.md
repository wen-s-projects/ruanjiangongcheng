# 卡卡项目 - 启动说明

## 🚀 快速启动

### 方法1：使用批处理文件（推荐）

**启动后端：**
1. 双击运行：`启动后端服务器.bat`
2. 等待看到：`Starting development server at http://127.0.0.1:8000/`

**启动前端：**
1. 双击运行：`启动前端.bat`
2. 等待看到：`Local: http://localhost:5173/`（或5174、5175）

---

### 方法2：手动启动

**启动后端（Django）：**
打开CMD（不是PowerShell），依次执行：
```bash
cd backend\calorie_backend
python manage.py runserver 0.0.0.0:8000
```

**启动前端（Vue 3）：**
打开新的CMD窗口，依次执行：
```bash
cd frontend
npm run dev
```

## 🔧 故障排除

### 如果后端无法启动：

1. 确保Python已安装：`python --version`
2. 确保Django已安装：`python -m pip install Django`
3. 检查8000端口是否被占用：`netstat -ano | findstr :8000`
4. 查看错误日志

### 如果前端无法访问：

1. 确保npm已安装：`npm --version`
2. 重新安装依赖：`cd frontend && npm install`
3. 重新启动：`npm run dev`
